Eurisko / Glossario SAP

Glossario SAP.

Oltre 50 termini, acronimi e tecnologie del mondo SAP, spiegati senza fronzoli. Dai pilastri (ECC, S/4HANA, ABAP) alle integrazioni (IDOC, BAPI, CPI), dai moduli funzionali alle metodologie di migrazione. Curato dai consulenti Eurisko per chi inizia a lavorare con SAP — e per chi vuole sciogliere dubbi.

A B C D E F G H I L M N O P R S T U W

A

ABAP
Advanced Business Application Programming. Linguaggio di programmazione proprietario SAP, usato per customizzazioni, sviluppi su misura e integrazioni interne. È la base storica di gran parte del software SAP.
ALV
ABAP List Viewer. Tool standard SAP per visualizzare report tabellari interattivi (ordinamento, filtri, esportazione Excel) senza dover sviluppare un'interfaccia da zero.
Ariba
Piattaforma cloud SAP per il procurement: gestione fornitori, gare, contratti, ordini di acquisto, fatturazione elettronica. Spesso integrata con SAP S/4HANA.

B

BAPI
Business Application Programming Interface. API standard SAP che espone in modo controllato funzionalità di business (creare un ordine, registrare un movimento merce, ecc.) per integrazioni con altri sistemi.
Blueprint
Documento di design che descrive come i processi to-be saranno realizzati nel sistema SAP target. Output della fase di analisi, input per la fase di build.
Brownfield
Approccio di migrazione a S/4HANA in cui si converte tecnicamente il sistema esistente preservando configurazioni, dati e personalizzazioni. Approccio rapido se il sistema attuale è "pulito".
BTP
SAP Business Technology Platform. Piattaforma cloud SAP per estendere le applicazioni standard, costruire integrazioni, sviluppare app custom e usare servizi di analytics, AI e database HANA-as-a-service.
BW / BW/4HANA
Business Warehouse. Data warehouse SAP. BW/4HANA è la versione moderna basata su database HANA, ottimizzata per analytics in-memory.

C

CDS Views
Core Data Services. Tecnologia SAP per definire viste dati semantiche direttamente nel database HANA, alla base di Fiori, embedded analytics e applicazioni S/4HANA moderne.
CO
Controlling. Modulo SAP per la contabilità analitica: centri di costo, centri di profitto, ordini interni, calcolo dei costi di prodotto, profittabilità.
CPI
SAP Cloud Platform Integration (oggi parte di Integration Suite su BTP). Tool cloud per orchestrare integrazioni tra SAP e sistemi esterni. Successore moderno di PI/PO on-premise.
Customizing
Configurazione standard di SAP attraverso le tabelle parametriche IMG, senza scrivere codice. Da non confondere con sviluppi ABAP custom.

D

Data Migration
Processo di trasferimento dati anagrafici e transazionali dal sistema legacy a SAP. Include estrazione, pulizia, mapping, caricamento e riconciliazione.

E

ECC
ERP Central Component. La generazione di SAP ERP più diffusa nelle aziende italiane, in uso da metà anni 2000. Il supporto standard SAP termina il 31 dicembre 2027 — da qui la spinta verso S/4HANA.
ERP
Enterprise Resource Planning. Categoria di software che integra in un'unica suite la gestione di tutti i processi aziendali core. SAP è il principale fornitore mondiale.
EWM
Extended Warehouse Management. Soluzione SAP avanzata per la gestione del magazzino: movimentazione, picking, packing, gestione layout di magazzino complessi. Successore moderno di SAP WM.

F

FI
Financial Accounting. Modulo SAP per la contabilità generale: contabilità clienti, fornitori, cespiti, banche, bilancio e adempimenti civilistici e fiscali.
FI-CA
Contract Accounting. Estensione di FI per la contabilità clienti su volumi massivi: utility, telco, servizi pubblici. Gestisce milioni di utenze e fatturazioni periodiche.
Fiori
Linguaggio di design e libreria di app SAP a interfaccia moderna, role-based, mobile-friendly. Sostituisce progressivamente le vecchie SAP GUI nelle attività degli utenti finali.

G

Go-live
Momento in cui il nuovo sistema SAP entra in produzione e gli utenti iniziano a usarlo per le operazioni reali. Tipicamente preceduto da un periodo di freeze e seguito da un periodo di hypercare.
GRC
Governance, Risk, Compliance. Suite SAP per gestire ruoli e autorizzazioni, segregation of duties, audit, controlli interni e conformità normativa.
Greenfield
Approccio di migrazione a S/4HANA in cui si implementa il nuovo sistema da zero, sfruttando i processi standard SAP più moderni. Ideale se il sistema attuale ha molte customizzazioni obsolete.

H

HANA
Database in-memory SAP, alla base di S/4HANA. Combina elaborazione transazionale e analitica nello stesso sistema, con performance molto superiori ai database tradizionali.
HCM
Human Capital Management. Modulo SAP per HR, payroll e gestione del personale on-premise. La versione cloud è SuccessFactors.
Hypercare
Periodo di supporto intensivo (tipicamente 4–8 settimane) successivo al go-live, in cui il team di progetto resta vicino agli utenti per stabilizzare il sistema.

I

IBP
Integrated Business Planning. Soluzione cloud SAP per la pianificazione integrata della supply chain: domanda, scorte, produzione, sales & operations planning.
IDOC
Intermediate Document. Formato standard SAP per scambiare dati strutturati tra sistemi (es. ordini, fatture, anagrafiche). Veicolato tipicamente via PI/PO o CPI.
IS-U
Industry Solution Utilities. Soluzione SAP verticale per le utility (luce, gas, acqua): anagrafica clienti, contratti di fornitura, device management, fatturazione massiva.

L

LSMW
Legacy System Migration Workbench. Tool SAP storico per il caricamento di dati anagrafici da file legacy. In progetti S/4HANA è progressivamente sostituito da LTMC e Migration Cockpit.

M

MES (integrazione)
Manufacturing Execution System. Sistema di esecuzione produzione, esterno a SAP ma frequentemente integrato con i moduli PP/QM per scambiare ordini di produzione, dichiarazioni di consuntivo e dati di qualità.
MM
Material Management. Modulo SAP per acquisti, gestione materiali, anagrafica fornitori, ordini di acquisto, ricezione merci, fatturazione passiva.

N

Note OSS / SAP Notes
Pubblicazioni ufficiali SAP che descrivono bug, fix, raccomandazioni, istruzioni di applicazione di patch. Distribuite tramite il portale di supporto SAP.

O

OData
Open Data Protocol. Standard REST per esporre dati strutturati. SAP lo usa per servire i dati alle app Fiori e a integrazioni moderne basate su API web.

P

PI / PO
Process Integration / Process Orchestration. Middleware SAP on-premise per integrare sistemi (legacy, MES, CRM, ecc.) con SAP. Successore: SAP CPI / Integration Suite.
PM
Plant Maintenance. Modulo SAP per la manutenzione di impianti e macchinari: anagrafica asset, piani di manutenzione preventiva, ordini di lavoro, gestione interventi.
PP
Production Planning. Modulo SAP per la pianificazione e l'esecuzione della produzione: distinte base, cicli di lavoro, MRP, ordini di produzione.

R

RFC
Remote Function Call. Protocollo SAP per invocare funzioni ABAP da un sistema esterno o da un altro sistema SAP. Alla base di molte integrazioni interne.
RICEFW
Reports / Interfaces / Conversions / Enhancements / Forms / Workflows. Tassonomia classica per classificare gli sviluppi custom in un progetto SAP, utile per stimare effort e governance.
RISE with SAP
Offerta commerciale SAP che bundleizza S/4HANA Cloud, infrastruttura, servizi di trasformazione e Business Network in un unico contratto. Versione "GROW with SAP" è dedicata alle PMI.

S

S/4HANA
La generazione attuale di SAP ERP, basata sul database in-memory HANA. Disponibile in versione on-premise, private cloud (RISE) e public cloud. È il successore di ECC.
SAC
SAP Analytics Cloud. Soluzione cloud per business intelligence, planning e predictive analytics. Sostituisce progressivamente strumenti come BusinessObjects e Lumira.
SD
Sales & Distribution. Modulo SAP per il ciclo attivo: anagrafica clienti, offerte, ordini di vendita, spedizioni, fatturazione.
SDT
Selective Data Transition. Approccio ibrido di migrazione a S/4HANA: si scelgono quali dati e processi portare nel nuovo sistema e cosa ripensare. Compromesso tra Greenfield e Brownfield.
SLA
Service Level Agreement. Contratto che definisce i livelli di servizio garantiti (tempi di presa in carico, risoluzione, uptime) tipicamente nei contratti AMS.
SuccessFactors
Suite cloud SAP per HR moderno: recruiting, performance, learning, compensation, employee experience. Successore cloud di SAP HCM on-premise.

T

T-code
Transaction Code. Codice alfanumerico che identifica una transazione SAP (es. VA01 = creazione ordine di vendita, FB60 = registrazione fattura fornitore).

U

UI5 / SAPUI5
Framework JavaScript SAP per costruire app web responsive (Fiori si basa su UI5). Equivalente, in casa SAP, a React o Angular.

W

WM
Warehouse Management. Modulo SAP "classico" per la gestione magazzino. Per esigenze più avanzate, è stato superato da EWM.
Workflow
Motore SAP per automatizzare processi che richiedono approvazioni e step manuali (es. autorizzazione di un ordine, approvazione fattura, escalation).
